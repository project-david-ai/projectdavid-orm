# Security Policy

## Supported Versions

We currently only provide security updates for the latest stable release of `projectdavid`. Please ensure you are running the latest version from PyPI before reporting a vulnerability.
# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| >= 1.1.10 | ✅ |
| < 1.1.10 | ❌ |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report security issues privately to:

**Email:** [engineering@projectdavid.co.uk](mailto:engineering@projectdavid.co.uk)

Please include:
- A description of the vulnerability
- Steps to reproduce
- The version of `projectdavid-orm` you are running

You will receive acknowledgment within 48 hours. Please do not disclose the issue publicly until a patched version has been released.


## Responsible Disclosure

We are a small team (currently a solo maintainer). We appreciate your patience and your help in keeping the `projectdavid` ecosystem safe for our 280k+ users.
